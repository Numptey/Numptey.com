Description

A mod created to break the trend of horror mods that contain re-textured zombies with creepy sounds that this game has been getting for the past few years.

Features

  • Mobs use ChatGPT to "think" in real time
  • Uses the player's name on their Computer
  • Uses the player's IP address
  • Takes screenshots
  • Creates files on the player's computer (.txt, and .bat)
  • Malware/Virus visuals drawn directly on the player's desktop
  • Faux Blue Screen of Death
  • Forcefully powers off the player's computer

How Does It Work?

How do the mobs "think" in real time?

Certain environmental factors are taken and sent to GPT 4o Mini using OpenAI's API Platform. ChatGPT uses the information provided to make it's decision. Once the decision is made it gets sent back to the mod, and it's acted out. This is also how the "Chat Monster" works minus the environmental factors being taken in

How does the mod's shader work?

When the sun reaches a certain position in the sky a custom fog effect is applied to everything. The fog effect works by absorbing light. When the night is beginning it starts with a low absorption percentage but gradually increases as the night goes on until pitch darkness (100% absorption). Light sources in the darkness lessen the nearby fogs absorption, but the further you are from a light source the less it effects absorption

How does the mod use real life info such as the player's IP Address?

The mod uses the website "https://api.ipify.org" to geolocate the player

How does the mod work outside the game?

The mod uses a well known program called AHK (Auto Hot Key) which does the minimization process, and fake errors as well as chaining everything together (the first AHK script is executed via Java then right before that script ends it executes the next script and this sequence repeats itself until the process is over

How do the virus visuals work?

A python executable is executed by an AHK script then it just plays out the executable's code

How does the mod turn off the player's computer?

"Shutdown 13". That code is all it takes to turn off the player's computer

  • Windows 11 (Windows 10 and Windows 11 Pro should work too)
  • Minecraft Java Edition
  • Fabric Loader
